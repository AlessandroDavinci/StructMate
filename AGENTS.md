# StructMate 项目开发规范

## 项目目标

开发面向数据结构零基础计算机本科生的个性化学习多智能体Web系统。

## 固定技术栈

- Python 3.11
- Streamlit
- SQLite
- pytest
- OpenAI兼容格式的大模型接口
- Mermaid
- 简体中文界面
- 默认生成C++17代码案例

## 课程知识库

- 主知识库：course_materials/数据结构.md
- Markdown引用图片目录：course_materials/数据结构.assets
- 按Markdown标题进行章节切分和关键词检索
- 不使用向量数据库
- 不使用FAISS
- 不使用Embedding接口
- 暂不处理视频
- 不对图片进行OCR

## 系统架构

采用1个协调器和7个专业智能体：

1. 知识检索智能体
2. 教学设计智能体
3. 文档与实践资源生成智能体
4. 题库生成智能体
5. 学习路径规划智能体
6. 评估诊断智能体
7. 内容审核与防幻觉智能体

## 开发规则

- 优先保证系统可运行、可录屏演示
- 不开发教师端
- 不开发视频和动画生成
- 不开发在线代码执行沙箱
- 不采用复杂前后端分离架构
- 不擅自增加Redis、Docker或PostgreSQL
- API密钥只能从环境变量读取
- 无API密钥时必须支持演示缓存模式
- 不允许使用空函数冒充已实现功能
- 每完成一个阶段必须运行pytest
- 不得修改原始数据结构.md
- 发现课程资料疑似错误时记录到审核报告
